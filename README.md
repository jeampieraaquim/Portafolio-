# Quien Soy
Soy Jeampier Aquim, técnico en ciberseguridad con enfoque en hardening, pentesting web y análisis de vulnerabilidades.
Este repositorio reúne mis proyectos prácticos, laboratorios y documentación técnica, demostrando mis habilidades para roles como:

- SOC Analyst L1

- Junior Pentester

- Analista de Ciberseguridad

Mi objetivo es presentar un portafolio claro, técnico y orientado a escenarios reales.

# Habilidades 
- Linux Hardening (UFW, SSH secure, Fail2ban, Lynis)
- Pentesting Web (XSS, SQLi, CSRF, IDOR, File Upload)
- Automatización con Python
- Herramientas: Burp Suite, Nmap, Gobuster, Hydra, Nikto
- Virtualización con VirtualBox
